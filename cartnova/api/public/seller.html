API Endpoints used on this page
All endpoints use X-Seller-Key header authentication (not JWT)
GET /api/v2/sellers/me/products — product listings
POST /api/v2/sellers/me/products — create listing
PUT /api/v2/sellers/me/products/{id} — update listing
GET /api/v2/sellers/me/analytics — sales dashboard
GET /api/v2/sellers/me/orders — orders for my products
